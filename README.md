# Daily Planner

A simple Daily Planner application built to practice state management and component architecture.

## Features
- Add task
- Toggle completion
- Delete task
- Persistent data using localStorage

## Tech Stack
- React
- Custom Hooks
- CSS

## Learning Focus
- State ownership
- Separation of concerns
- useState vs useReducer exploration
- LocalStorage side-effect handling

1. Clone the repository


npm run dev
